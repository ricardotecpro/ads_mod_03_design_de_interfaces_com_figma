# Quiz 13: Bibliotecas e Design Systems

1. Um Design System é definido como:
    - [ ] Um conjunto de ícones coloridos.
    - [x] Um ecossistema vivo de componentes, estilos, padrões e documentação que guia a criação de produtos consistentes.
    - [ ] Um curso de desenho.

2. O que são "Tokens de Design"?
    - [ ] Moedas virtuais do Figma.
    - [x] Valores abstratos e semânticos que armazenam decisões de design (como cores e tamanhos) e podem ser usados no código.
    - [ ] Nomes de camadas aleatórios.

3. Para que serve a função "Publish Library" no Figma?
    - [ ] Para imprimir o design em papel.
    - [x] Para disponibilizar componentes e estilos para serem usados em outros arquivos da equipe.
    - [ ] Para deletar o projeto.

4. Qual a diferença entre um UI Kit e um Design System?
    - [ ] Não há diferença.
    - [x] Um UI Kit é apenas uma coleção de elementos visuais; um Design System inclui regras, documentação e processos.
    - [ ] Um é pago e o outro é gratuito.

5. Imagens e ícones de alta qualidade de bancos como Unsplash ou Lucide Icons são:
    - [ ] Proibidos por lei.
    - [x] Elementos de apoio essenciais que aceleram o fluxo de trabalho e trazem realismo ao design.
    - [ ] Usados apenas por amadores.

6. O termo "Single Source of Truth" (Fonte Única da Verdade) no contexto de Design Systems significa:
    - [ ] Que o chefe sempre tem razão.
    - [x] Que existe um único lugar centralizado onde as definições oficiais de design estão documentadas e atualizadas.
    - [ ] Que o arquivo original foi perdido.

7. Por que a documentação é crucial em um Design System?
    - [ ] Para o arquivo ficar mais pesado.
    - [x] Para que os usuários saibam como, quando e por que usar cada componente e estilo.
    - [ ] Para ganhar nota na escola.

8. Quando uma biblioteca publicada é atualizada, os arquivos que a utilizam:
    - [ ] São deletados.
    - [x] Recebem uma notificação para revisar as atualizações e aceitá-las.
    - [ ] Mudam automaticamente sem aviso.

9. "Accessibility Standards" (Padrões de Acessibilidade) devem estar presentes em:
    - [ ] Apenas em sites do governo.
    - [x] Em todos os fundamentos do Design System (cores, fontes, componentes).
    - [ ] Em lugar nenhum.

10. Um Design System ajuda na escalabilidade do produto porque:
    - [ ] Reduz o tamanho das imagens.
    - [x] Permite que novas telas sejam criadas rapidamente mantendo a consistência visual e técnica.
    - [ ] Aumenta o número de curtidas no LinkedIn.
