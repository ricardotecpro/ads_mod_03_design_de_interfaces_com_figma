# Quiz 15: Handover e Documentação

1. O que é o "Handover" no processo de design?
    - [ ] O momento de criar o logotipo.
    - [x] A passagem do design finalizado e suas especificações para a equipe de desenvolvimento.
    - [ ] O ato de deletar o arquivo original.

2. O "Dev Mode" (Modo de Desenvolvedor) do Figma serve para:
    - [ ] Programar em Python dentro do Figma.
    - [x] Facilitar a inspeção de propriedades de código (CSS, Swift, Kotlin) por desenvolvedores.
    - [ ] Jogar videogame.

3. Ao exportar um ícone para web, qual o formato mais recomendado por ser vetorial e leve?
    - [ ] JPG.
    - [x] SVG.
    - [ ] BMP.

4. O que significa exportar um asset em "2x" ou "3x"?
    - [ ] Exportar o arquivo duas vezes.
    - [x] Gerar a imagem com o dobro ou triplo da resolução para telas de alta densidade (Retina).
    - [ ] Aumentar o preço do projeto.

5. Qual a importância das anotações (Redlines) no Handover?
    - [ ] Decorar a tela.
    - [x] Explicar comportamentos complexos, fluxos e especificações que não são óbvias apenas olhando o design.
    - [ ] Esconder erros.

6. No painel de inspeção, o que o desenvolvedor pode ver ao clicar em um texto?
    - [ ] O histórico médico do designer.
    - [x] A família da fonte, tamanho, peso, line-height e cor em código CSS/Sass.
    - [ ] O preço da licença do Figma.

7. Por que devemos enviar um link de visualização em vez de apenas prints da tela?
    - [ ] Porque print é proibido.
    - [x] Para que o desenvolvedor possa inspecionar medidas exatas, cores e exportar assets diretamente.
    - [ ] Para ocupar menos espaço no e-mail.

8. O termo "Export Settings" permite definir:
    - [ ] A música de fundo.
    - [x] O formato (PNG, JPG, SVG, PDF) e a escala da imagem que será baixada.
    - [ ] O nome do computador.

9. No Handover, o designer deve se preocupar com quais estados de um botão?
    - [ ] Apenas o estado normal.
    - [x] Todos os estados interativos (Hover, Active, Disabled, Loading).
    - [ ] Apenas a cor de fundo.

10. Plugins de "Figma to Code" ajudam a:
    - [ ] Deletar o projeto.
    - [x] Transformar partes do design em código HTML/CSS ou frameworks prontos para uso.
    - [ ] Mudar a senha do Wi-Fi.
