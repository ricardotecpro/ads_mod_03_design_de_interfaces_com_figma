# Quiz 07: Imagens Raster (Bitmap)

1. Qual a principal característica de uma imagem Raster (Bitmap)?
    - [ ] É baseada em vetores matemáticos.
    - [x] É composta por uma grade de pixels.
    - [ ] Nunca perde qualidade ao ser ampliada.

2. O que acontece ao ampliar muito uma imagem Raster de baixa resolução?
    - [ ] Ela fica mais colorida.
    - [x] Ocorre a "pixelização" (a imagem fica borrada ou serrilhada).
    - [ ] Ela se transforma em vetor automaticamente.

3. No Figma, o que são "Máscaras" de imagem?
    - [ ] Filtros decorativos.
    - [x] Uma forma de ocultar partes de uma imagem usando outra forma como molde, de forma não-destrutiva.
    - [ ] Formas de esconder o nome das camadas.

4. Qual formato de imagem suporta transparência (canal alpha)?
    - [ ] JPG.
    - [x] PNG.
    - [ ] BMP.

5. Onde você ajusta brilho, contraste e saturação de uma imagem no Figma?
    - [ ] No painel de Camadas.
    - [x] Abrindo as configurações de Fill (Preenchimento) da imagem.
    - [ ] No menu de Ajuda.

6. O "Canal Alpha" em uma imagem refere-se a:
    - [ ] A qualidade da foto.
    - [x] O nível de transparência.
    - [ ] O tamanho do arquivo.

7. Qual ferramenta Open Source é recomendada para retoques avançados em fotos antes de importá-las para o Figma?

    - [x] GIMP.

8. "Krita" é um software Open Source focado em:
    - [ ] Edição de texto.
    - [x] Pintura digital e ilustrações raster.

9. Imagens muito pesadas no Figma podem causar:
