# Quiz 09: Responsividade (Resizing)

1. A configuração "Hug Contents" (Abraçar Conteúdo) faz com que o frame:
    - [ ] Fique sempre do mesmo tamanho.
    - [x] Encolha ou cresça para ficar do tamanho exato do que está dentro dele.
    - [ ] Estique até as bordas da tela.

2. A configuração "Fill Container" (Preencher Recipiente) faz com que o elemento:
    - [ ] Tenha um tamanho fixo definido por você.
    - [x] Se estique para ocupar todo o espaço disponível dentro do frame pai (respeitando o padding).
    - [ ] Desapareça da tela.

3. Quando usamos "Fixed Width", o elemento:
    - [ ] Muda de tamanho conforme o texto.
    - [x] Mantém a largura exata definida, ignorando o tamanho do conteúdo ou do frame pai.
    - [ ] Fica invisível.

4. O recurso "Wrap" (Quebra de Linha) no Auto Layout é útil para:
    - [ ] Deletar camadas.
    - [x] Fazer com que itens "pulem" para a linha de baixo quando não houver mais espaço lateral.
    - [ ] Girar o objeto em 360 graus.

5. Qual a combinação ideal para um botão que deve crescer conforme o texto?
    - [ ] Frame com largura Fixa.
    - [x] Frame com configuração "Hug Contents" na largura e altura.
    - [ ] Frame com "Fill Container" e texto fixo.

6. "Min-width" e "Max-width" são usados para:
    - [ ] Aumentar a resolução da imagem.
    - [x] Definir limites mínimos e máximos de largura para que o elemento não fique nem muito pequeno nem muito grande.
    - [ ] Mudar a fonte.

7. Interface responsiva é aquela que:
    - [ ] Responde a comandos de voz.
    - [x] Se adapta visualmente a diferentes tamanhos de tela (desktop, tablet, mobile).
    - [ ] Carrega em menos de 1 segundo.

8. Ao projetar para Mobile, qual a largura padrão mais comum usada no Figma para iPhones modernos?
    - [ ] 1920px.
    - [x] 390px ou 375px.
    - [ ] 100px.

9. A propriedade "Absolute Position" dentro de um frame de Auto Layout permite:
    - [ ] Quebrar o arquivo.
    - [x] Que um elemento ignore as regras do Auto Layout e seja posicionado livremente no frame.
    - [ ] Mudar o nome do projeto.

10. Se um card de notícia deve ocupar toda a largura da tela do celular, qual redimensionamento devemos usar nele?
    - [ ] Hug Contents.
    - [x] Fill Container.
    - [ ] Fixed.
