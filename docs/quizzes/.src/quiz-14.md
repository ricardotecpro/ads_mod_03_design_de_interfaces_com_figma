# Quiz 14: Prototipagem e Interatividade

1. Qual o objetivo principal de um protótipo no Figma?
    - [ ] Exportar o código final do aplicativo.
    - [x] Simular a experiência de uso e o fluxo de navegação do produto final.
    - [ ] Fazer uma apresentação de slides estática.

2. No modo de Prototipagem, o que representam os "Nós" e "Setas Azuis"?
    - [ ] Erros de conexão de internet.
    - [x] Conexões que definem o ponto de partida (trigger) e o destino da interação.
    - [ ] Decoração do canvas.

3. O que é um "Trigger" (Gatilho) na prototipagem?
    - [ ] A cor do botão.
    - [x] A ação do usuário que inicia a animação (ex: On Click, While Hovering).
    - [ ] O nome da página.

4. A animação "Smart Animate" funciona:
    - [ ] Apenas em imagens JPG.
    - [x] Identificando camadas com o mesmo nome em telas diferentes e animando as transições entre elas.
    - [ ] Traduzindo os textos para inglês.

5. O que acontece na ação "Open Overlay"?
    - [ ] Uma nova página é carregada por completo.
    - [x] Um elemento (como um menu ou modal) aparece sobre a tela atual.
    - [ ] O Figma fecha o arquivo.

6. "A fidelidade" de um protótipo refere-se a:
    - [ ] O quanto o designer é fiel à empresa.
    - [x] O nível de detalhamento visual e de interatividade (Baixa vs Alta fidelidade).
    - [ ] O preço do software.

7. O atalho para entrar no modo de Visualização (ver o protótipo rodando) é:
    - [ ] Ctrl + S.
    - [x] O ícone de "Play" no canto superior direito.
    - [ ] Alt + F4.

8. Para criar um efeito de "Scroll" horizontal em uma lista, você deve:
    - [ ] Deletar a lista.
    - [x] Colocá-la em um frame com Auto Layout e mudar o "Overflow" para horizontal no painel de prototipagem.
    - [ ] Mudar a fonte para itálico.

9. Animações de "Ease In" e "Ease Out" servem para:
    - [ ] Deixar as cores mais fortes.
    - [x] Dar um movimento mais natural e suave às transições, simulando aceleração física.
    - [ ] Economizar bateria.

10. No Figma, podemos testar protótipos diretamente no celular?
    - [x] Sim, usando o aplicativo "Figma Mirror" ou enviando o link do protótipo.
    - [ ] Não, apenas no computador.
    - [ ] Apenas se o celular for um iPhone.
