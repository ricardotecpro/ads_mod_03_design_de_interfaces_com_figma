# Quiz 08: Auto Layout Fundamental

1. O que é o Auto Layout no Figma?
    - [ ] Uma ferramenta de desenho à mão livre.
    - [x] Um recurso que permite criar frames dinâmicos que se ajustam automaticamente ao conteúdo (similar ao Flexbox).
    - [ ] Um plugin de tradução de texto.

2. Qual o atalho para adicionar Auto Layout a uma seleção?
    - [ ] Ctrl + G.
    - [x] Shift + A.
    - [ ] Alt + F4.

3. O parâmetro "Gap" ou "Spacing between items" define:
    - [ ] O tamanho da fonte.
    - [x] A distância constante entre os elementos dentro do frame de Auto Layout.
    - [ ] A cor de fundo.

4. O que é o "Padding" no Auto Layout?
    - [ ] O espaço entre dois objetos diferentes.
    - [x] O espaço interno entre as bordas do frame e os elementos contidos nele.
    - [ ] O nome do autor do projeto.

5. Quais são as duas direções possíveis em um frame simples com Auto Layout?
    - [ ] Circular e Diagonal.
    - [x] Vertical (Coluna) e Horizontal (Linha).
    - [ ] Cima e Baixo apenas.

6. Ao usar Auto Layout em um botão, se você mudar o texto de "OK" para "Confirmar cadastro", o botão irá:
    - [ ] Cortar o texto.
    - [x] Crescer automaticamente para acomodar o texto maior.
    - [ ] Ficar com a cor vermelha.

7. O "Alignment" no Auto Layout permite posicionar os itens em quantas posições diferentes dentro do frame?
    - [ ] 2 (Esquerda e Direita).
    - [x] 9 (Topo-Esquerda, Centro, Base-Direita, etc.).
    - [ ] Infinitas.

8. Para remover um Auto Layout de um frame, qual o atalho?
    - [ ] Delete.
    - [x] Alt + Shift + A.
    - [ ] Ctrl + Z.

9. "Canvas Stacking" refere-se à ordem em que os elementos se sobrepõem quando o gap é:
    - [ ] Positivo.
    - [x] Negativo (elementos ficam um em cima do outro).
    - [ ] Zero.

10. Grupos (Ctrl+G) são diferentes de Auto Layout pois:
    - [ ] Grupos são pagos.
    - [x] Grupos apenas agrupam, enquanto Auto Layout impõe regras de alinhamento e espaçamento dinâmico.
    - [ ] Não há diferença real.
