# Quiz 06: Estilos Globais

1. Para que servem os estilos no Figma?
    - [ ] Mudar a cor do ícone do aplicativo.
    - [x] Manter a consistência visual e facilitar atualizações globais em cores, textos e efeitos.
    - [ ] Criar desenhos à mão livre.

2. Se você alterar a cor de um "Color Style", o que acontece com as camadas vinculadas a ele?
    - [ ] Nada acontece.
    - [x] Todas as camadas vinculadas são atualizadas automaticamente com a nova cor.
    - [ ] Elas perdem a conexão com o estilo.

3. Qual o limitador usado no nome do estilo para criar hierarquia de pastas (ex: Brand / Primary)?
    - [ ] Ponto (.).
    - [x] Barra diagonal (/).
    - [ ] Arroba (@).

4. "Text Styles" no Figma permitem salvar propriedades de:
    - [ ] Apenas a cor do texto.
    - [x] Fonte, tamanho, peso, altura de linha e espaçamento.
    - [ ] A imagem de fundo.

5. Qual o ícone que representa os estilos globais no painel lateral?
    - [ ] Um círculo preenchido.
    - [x] Um ícone de quatro pontos formando um quadrado (Grid).
    - [ ] Uma estrela.

6. O que são "Effect Styles"?
    - [ ] Filtros do Instagram.
    - [x] Estilos salvos de sombras (Drop Shadow), brilhos ou desfoques (Blur).
    - [ ] Sombras de árvores no design.

7. No Figma, como você desconecta um elemento de um estilo global?
    - [ ] Deletando o elemento.
    - [x] Clicando no ícone de "Detach Style" (elo quebrado) no painel de propriedades.
    - [ ] Reiniciando o computador.

8. Por que é melhor nomear um estilo como "Success" em vez de "Green"?
    - [ ] Para parecer mais profissional.
    - [x] Pois se a cor de sucesso mudar para azul no futuro, o nome continua fazendo sentido semanticamente.
    - [ ] O Figma não aceita nomes de cores.

9. "Grid Styles" permitem salvar configurações de:
    - [ ] Apenas a cor da tela.
    - [x] Colunas, linhas e grades de layout.
    - [ ] O tamanho da fonte.

10. Estilos Globais são essenciais para a criação de um:
    - [ ] Arquivo temporário.
    - [x] Design System.
    - [ ] Logotipo simples.
