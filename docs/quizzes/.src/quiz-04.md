# Quiz 04: Tipografia e Hierarquia Visual

1. Qual o termo usado para descrever a ordenação estratégica de elementos para guiar o olhar do usuário?
    - [ ] Caos visual.
    - [x] Hierarquia Visual.
    - [ ] Pintura Digital.

2. Fontes do tipo "Sans Serif" são conhecidas por:
    - [ ] Terem "serifas" (pequenos traços nas pontas).
    - [x] Não possuírem serifas, tendo um aspecto mais moderno e limpo.
    - [ ] Serem escritas à mão.

3. O que é o "Line-height" (Altura de Linha)?
    - [ ] A largura da letra.
    - [x] O espaço vertical entre as linhas de um parágrafo.
    - [ ] A cor do texto.

4. Para destacar uma informação em um texto sem mudar o tamanho da fonte, você deve alterar o:
    - [ ] Nome do arquivo.
    - [x] Peso (Weight), como colocar em Negrito (Bold).
    - [ ] Brilho do monitor.

5. O que significa "Letter-spacing" ou "Kerning"?
    - [ ] Espaço entre as linhas.
    - [x] Espaço horizontal entre as letras de uma palavra.
    - [ ] O tamanho da página.

6. Segundo as boas práticas, quantas fontes diferentes (famílias) devemos usar em um projeto padrão?
    - [ ] No mínimo 10.
    - [x] Evitar passar de 2 ou 3 para manter a consistência.
    - [ ] Apenas fontes cursivas.

7. No Figma, em qual painel encontramos as configurações de texto?
    - [ ] Painel de Camadas.
    - [x] Painel de Propriedades (Design), seção Text.
    - [ ] Barra de Favoritos.

8. Qual a diferença entre Font e Typeface?
    - [ ] São a mesma coisa sempre.
    - [x] Typeface é a família (ex: Roboto), Font é o arquivo específico (ex: Roboto Bold 12pt).
    - [ ] Uma é paga e a outra é gratuita.

9. O que é "Legibilidade"?
    - [ ] A beleza intrínseca da fonte.
    - [x] A facilidade com que o cérebro reconhece e lê os caracteres.
    - [ ] A velocidade da internet.

10. Usar apenas letras maiúsculas (UPPERCASE) em parágrafos longos é:
    - [ ] Recomendado para dar ênfase.
    - [x] Desaconselhado, pois dificulta a leitura e cansa o usuário.
    - [ ] Obrigatório em sites do governo.
