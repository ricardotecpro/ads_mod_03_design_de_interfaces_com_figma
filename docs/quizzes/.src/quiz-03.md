# Quiz 03: Vetores e Open Source

1. Qual a principal vantagem dos vetores sobre imagens raster (bitmap)?
    - [ ] São mais coloridos.
    - [x] Podem ser redimensionados infinitamente sem perda de qualidade.
    - [ ] São arquivos sempre maiores.

2. No Figma, qual ferramenta é usada para criar e manipular caminhos vetoriais livres?
    - [ ] Rectangle Tool.
    - [x] Pen Tool (P).
    - [ ] Hand Tool.

3. Os vetores são baseados em:
    - [ ] Pixels coloridos aleatoriamente.
    - [x] Cálculos matemáticos de pontos e curvas.
    - [ ] Fitas magnéticas.

4. Qual destas é uma operação Booleana (Boolean Operation)?
    - [x] Union (União).
    - [ ] Multiplication.
    - [ ] Division.

5. O que a operação "Subtract" faz entre duas formas?
    - [ ] Muda a cor de ambas.
    - [x] Remove a forma de cima da forma de baixo.
    - [ ] Cria um grupo.

6. Qual tecla você deve segurar para criar um círculo ou quadrado perfeito?
    - [ ] Ctrl.
    - [x] Shift.
    - [ ] Alt.

7. Qual ferramenta Open Source é considerada o "Illustrator" do mundo livre?
    - [ ] GIMP.
    - [x] Inkscape.
    - [ ] Notepad++.

8. O que é o Penpot?
    - [ ] Uma ferramenta de edição de vídeo.
    - [x] Uma alternativa Open Source ao Figma para design e prototipagem via web.
    - [ ] Um plugin para limpeza de arquivos.

9. "Curvas de Bézier" são usadas para:
    - [ ] Ajustar o volume do som.
    - [x] Manipular a curvatura entre dois pontos de ancoragem em um vetor.
    - [ ] Calcular o imposto de renda.

10. Para que serve a operação "Intersect"?
    - [ ] Deleta tudo da tela.
    - [x] Mantém apenas a área onde as formas selecionadas se sobrepõem.
    - [ ] Inverte as cores.
