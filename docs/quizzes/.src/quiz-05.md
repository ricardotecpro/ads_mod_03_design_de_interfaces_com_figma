# Quiz 05: Teoria das Cores e Acessibilidade

1. Qual modelo de cor é o padrão para telas digitais (monitores e celulares)?
    - [ ] CMYK.
    - [x] RGB.
    - [ ] Panton.

2. O que representa um código HEX de cor (ex: #FFFFFF)?
    - [ ] O custo da cor.
    - [x] Um valor hexadecimal que define a quantidade de Vermelho, Verde e Azul.
    - [ ] A data de criação.

3. Cores "Complementares" são aquelas que estão:
    - [x] Em lados opostos no círculo cromático.
    - [ ] Lado a lado no círculo cromático.
    - [ ] Na mesma pasta.

4. Qual esquema de cores usa apenas uma cor base com diferentes tons, brilhos e saturações?
    - [ ] Complementar.
    - [x] Monocromático.
    - [ ] Triádico.

5. Na psicologia das cores, o Azul costuma ser associado a:
    - [ ] Perigo e urgência.
    - [x] Confiança, segurança e calma.
    - [ ] Fome e energia.

6. O que é o "Contraste" no design de interface?
    - [ ] A beleza das cores juntas.
    - [x] A diferença de luminância entre o texto e o fundo.
    - [ ] A velocidade da animação.

7. O que são os padrões WCAG?
    - [ ] Regras de como programar em Python.
    - [x] Diretrizes de acessibilidade para conteúdo web, incluindo contraste mínimo.
    - [ ] Um tipo de fonte gratuita.

8. A regra "60-30-10" sugere que:
    - [ ] Devemos gastar 60 reais em cores.
    - [x] 60% cor primária (neutra), 30% secundária e 10% cor de destaque (CTA).
    - [ ] O site deve carregar em 60 segundos.

9. "Saturação" refere-se a:
    - [ ] O brilho da cor.
    - [x] A intensidade ou pureza de uma cor.
    - [ ] O nome da ferramenta de pintura.

10. Por que não devemos usar APENAS cores para transmitir informação (ex: verde para sucesso e vermelho para erro)?
    - [ ] Porque fica feio.
    - [x] Para garantir acessibilidade a usuários daltônicos (devemos usar ícones ou textos junto).
    - [ ] Porque é proibido por lei.
