# Quiz 10: Constraints e Grids

1. Para que servem as "Constraints" (Restrições) no Figma?
    - [ ] Para limitar o número de camadas.
    - [x] Definir como os elementos devem se comportar quando o frame pai é redimensionado (fixar em bordas ou esticar).
    - [ ] Bloquear a edição por outros usuários.

2. Se você aplicar a Constraint "Center" tanto na horizontal quanto na vertical, o objeto ficará:
    - [ ] No canto superior esquerdo.
    - [x] Sempre centralizado em relação ao frame pai, independente do tamanho deste.
    - [ ] Esticado até as bordas.

3. Para manter um logotipo sempre no topo e à esquerda de um site, as Constraints devem ser:
    - [ ] Right e Bottom.
    - [x] Left e Top.
    - [ ] Scale.

4. A Constraint "Scale" faz com que o objeto:
    - [ ] Mantenha o tamanho fixo centralizado.
    - [x] Cresça ou diminua proporcionalmente ao tamanho do frame pai.
    - [ ] Desapareça.

5. O que são "Layout Grids"?
    - [ ] Filtros para fotos.
    - [x] Orientadores visuais (colunas ou grades) que ajudam a alinhar elementos com precisão.
    - [ ] Listas de tarefas.

6. Em design para Web Desktop, qual o número de colunas mais utilizado tradicionalmente em sistemas como o Bootstrap?
    - [ ] 5 colunas.
    - [x] 12 colunas.
    - [ ] 100 colunas.

7. O "Gutter" em um Layout Grid de colunas refere-se a:
    - [ ] A largura da coluna.
    - [x] O espaço vazio entre as colunas.
    - [ ] A margem externa.

8. O "Grid de 8 pontos" (8pt Grid System) é uma recomendação para que:
    - [ ] Todos os textos tenham 8 pixels.
    - [x] Todos os espaçamentos, tamanhos e margens sejam múltiplos de 8.
    - [ ] Você use apenas 8 cores.

9. Quando um Layout Grid está configurado como "Stretch" (Esticar):
    - [ ] Ele tem largura fixa centralizada.
    - [x] As colunas mudam de largura conforme você redimensiona o frame.
    - [ ] Ele fica invisível.

10. Constraints só funcionam se os elementos estiverem:
    - [ ] Fora de qualquer frame.
    - [x] Dentro de um Frame (não funcionam em Grupos do tipo Ctrl+G de forma nativa avançada).
    - [ ] Na cor preta.
