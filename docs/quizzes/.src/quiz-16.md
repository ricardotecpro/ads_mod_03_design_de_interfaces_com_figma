# Quiz 16: Portfólio e Carreira

1. Qual o principal objetivo de um portfólio de UI Design?
    - [ ] Mostrar que você sabe usar todos os softwares do mundo.
    - [x] Demonstrar sua habilidade em resolver problemas e seu processo de design para potenciais empregadores.
    - [ ] Ganhar seguidores no Instagram.

2. Um "Case Study" (Estudo de Caso) deve conter:
    - [ ] Apenas as imagens finais bonitas.
    - [x] O desafio inicial, o processo de pesquisa, as iterações e o resultado final fundamentado.
    - [ ] A sua lista de compras.

3. Qual plataforma é mundialmente conhecida como a vitrine principal para trabalhos visuais detalhados?
    - [ ] TikTok.
    - [x] Behance.
    - [ ] WhatsApp.

4. Por que é importante incluir "Mockups" no seu portfólio?
    - [ ] Para o arquivo ficar mais colorido.
    - [x] Para dar contexto real ao seu design, mostrando como ele ficaria em um iPhone, Android ou Mac.
    - [ ] Porque é obrigatório no Figma.

5. Qual rede social é a mais indicada para networking profissional e busca de emprego em design?
    - [ ] Snapchat.
    - [x] LinkedIn.
    - [ ] Pinterest.

6. No seu portfólio, a qualidade é mais importante que a quantidade?
    - [x] Sim, é melhor ter 3 projetos excelentes do que 20 projetos médios ou incompletos.
    - [ ] Não, quanto mais projetos, melhor.
    - [ ] Depende do dia.

7. O que os recrutadores buscam ver além das telas finais?
    - [ ] O que você comeu no almoço.
    - [x] Suas decisões de design, como você lidou com restrições e sua capacidade de comunicação.
    - [ ] Fotos da sua família.

8. Qual o papel do Dribbble na carreira de um designer?
    - [ ] Armazenar arquivos pesados.
    - [x] Atuar como uma vitrine de "shots" rápidos e inspiração visual instantânea.
    - [ ] Enviar e-mails marketing.

9. Manter-se atualizado na área de UI Design significa:
    - [ ] Aprender apenas uma ferramenta para sempre.
    - [x] Estudar novas tendências, atualizações de softwares e ler artigos de especialistas (UX Collective, Nielsen Norman, etc.).
    - [ ] Ver apenas vídeos engraçados.

10. Ao finalizar este curso, o primeiro passo para sua carreira deve ser:
    - [ ] Deletar tudo o que aprendeu.
    - [x] Praticar criando projetos fictícios ou reais e começar a documentar seu processo em um portfólio.
    - [ ] Esperar que alguém te ligue oferecendo emprego.
