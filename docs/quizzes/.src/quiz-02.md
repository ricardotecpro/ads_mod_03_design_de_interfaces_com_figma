# Quiz 02: Interface e Atalhos

1. Qual painel no Figma é usado para gerenciar páginas e camadas?
    - [x] Painel Esquerdo (Layers).
    - [ ] Painel Direito (Properties).
    - [ ] Barra Superior.

2. Onde você ajusta as propriedades de cor e tamanho de um elemento selecionado?
    - [ ] Painel de Camadas.
    - [x] Painel de Propriedades (Direito).
    - [ ] Barra Central.

3. Qual a função da Barra Superior no Figma?
    - [ ] Exibir apenas o nome do arquivo.
    - [x] Acessar ferramentas de desenho, formas, texto e componentes.
    - [ ] Configurar a conta do usuário.

4. O que acontece ao usar o atalho "Espaço + Clique e Arrastar"?
    - [ ] Deleta o objeto selecionado.
    - [x] Ativa o Pan (Mover a visualização do Canvas).
    - [ ] Cria um novo frame.

5. Qual o atalho para dar Zoom In no Figma?
    - [ ] Ctrl + Z.
    - [x] Ctrl + (+) ou Ctrl + Scroll para cima.
    - [ ] Alt + F4.

6. O que são os "Frames" no Figma?
    - [ ] Apenas molduras decorativas.
    - [x] As "telas" ou áreas de trabalho onde o design é construído.
    - [ ] Pastas locais no seu computador.

7. Por que é importante manter as camadas nomeadas?
    - [ ] Não é importante, o Figma organiza sozinho.
    - [x] Para facilitar a organização e o trabalho colaborativo (Handover).
    - [ ] Para o arquivo carregar mais rápido.

8. Qual o atalho para abrir a barra de busca rápida (Quick Action)?
    - [ ] Ctrl + S.
    - [x] Ctrl + /.
    - [ ] Shift + A.

9. O atalho "Alt + L" serve para:
    - [ ] Limpar o histórico.
    - [x] Recolher todas as camadas abertas no painel lateral.
    - [ ] Ligar a lanterna.

10. No Figma, o Canvas é a área:
    - [x] Central, onde o design acontece.
    - [ ] Lateral, onde ficam os plugins.
    - [ ] Superior, onde ficam as ferramentas.
