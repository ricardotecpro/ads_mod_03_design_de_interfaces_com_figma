# Quiz 11: Componentes e Instâncias

1. O que é um "Componente Principal" (Main Component) no Figma?
    - [ ] Uma cópia de um elemento.
    - [x] O elemento mestre que define as propriedades visuais de todas as suas instâncias.
    - [ ] Um plugin de cores.

2. Como é identificado o ícone do Componente Principal no painel lateral?
    - [ ] Um losango único contornado.
    - [x] Quatro losangos pequenos formando um losango maior.
    - [ ] Um círculo vermelho.

3. Uma "Instância" é:
    - [x] Uma cópia de um componente principal que permanece ligada a ele.
    - [ ] Um arquivo separado no seu computador.
    - [ ] O mesmo que um grupo (Ctrl+G).

4. O que acontece se você mudar a cor do Componente Principal?
    - [ ] Nada acontece nas instâncias.
    - [x] Todas as instâncias vinculadas mudam para a nova cor automaticamente (se não houver overrides).
    - [ ] O Figma fecha sozinho.

5. O que são "Overrides"?
    - [ ] Erros no sistema.
    - [x] Alterações locais em uma instância (como mudar um texto ou uma imagem) que não afetam o componente mestre.
    - [ ] O nome do designer chefe.

6. Qual o atalho para transformar um elemento selecionado em um Componente?
    - [ ] Ctrl + Alt + Delete.
    - [x] Ctrl + Alt + K.
    - [ ] Shift + S.

7. Se você quiser que uma instância pare de ser atualizada pelo componente mestre, você deve:
    - [ ] Deletar a instância.
    - [x] Usar a função "Detach Instance" (Desvincular Instância).
    - [ ] Reiniciar o Figma.

8. Para onde vão os componentes criados no arquivo?
    - [ ] Pasta de Downloads.
    - [x] Para o painel "Assets" no lado esquerdo.
    - [ ] Para a lixeira.

9. Qual a vantagem de usar componentes em um projeto grande?
    - [ ] Fazer o computador rodar mais rápido.
    - [x] Manter a consistência e permitir atualizações em massa de elementos repetidos.
    - [ ] Nenhuma vantagem técnica.

10. Você pode criar um componente dentro de outro componente (Nesting)?
    - [x] Sim, e isso é fundamental para criar componentes complexos de forma organizada.
    - [ ] Não, o Figma proíbe essa prática.
    - [ ] Apenas se as cores forem iguais.
